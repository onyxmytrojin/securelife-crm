Samples folder

These two files are lightweight placeholder PDFs to allow the repo to include sample files for the extraction demo:

- LIC_sample_policy.pdf — placeholder; replace with a real LIC policy PDF
- HDFC_sample_policy.pdf — placeholder; replace with a real HDFC Life policy PDF

To replace with real PDFs, download the policy PDFs and overwrite these files, then run:

```bash
git add samples/
git commit -m "Add sample insurance PDFs"
```

Notes:
- I couldn't download real PDFs from the web in this environment. If you want, I can provide commands to fetch specific PDFs when you give permission or provide URLs.
- For extraction testing, use real policy PDFs from public sources or sample policies from insurers.
